Full Description
----------------
RetroC AI IDE brings the classic Borland Turbo C 3.0 experience to modern
Windows — and adds a powerful local AI assistant that runs 100% offline on
your device. No subscriptions, no cloud, no API keys. Your code never leaves
your machine.

✦ EVERYTHING IN THE FREE VERSION
  - Authentic retro blue/cyan DOS-style interface
  - Multi-tab editing with syntax highlighting for C89/C99
  - Bundled TCC (Tiny C Compiler) — compile and run instantly, no save required
  - Find & Replace, Go to Line, Find Procedure
  - Full menu bar with all classic Turbo C menus
  - Error navigation, configurable compiler, project management
  - TCC source code available at https://repo.or.cz/tinycc.git

✦ LOCAL AI ASSISTANT (paid exclusive)
  - Powered by Phi-3 mini (~2.3 GB) — runs entirely on YOUR machine
  - No internet connection required after the one-time model download
  - No API key, no subscription, no data collection

✦ ASK MODE
  - Ask any C programming question directly in the IDE
  - Select code → right-click → "Ask AI" for instant code analysis
  - Right-click any compiler error → "Explain Error with AI"
  - Responses include syntax-highlighted code with Copy and Insert buttons

✦ AGENT MODE
  - Tell the AI what to change: "add input validation", "fix the segfault"
  - AI responds with a diff view (red = removed, green = added)
  - One-click Apply applies the edit to your file and auto-compiles
  - One-click Reject discards the suggestion — you stay in control

✦ PRIVACY FIRST
  - Zero telemetry, zero analytics
  - All AI inference runs locally via llama.cpp / LLamaSharp
  - AI model stored in your AppData — never uploaded anywhere
  - No Microsoft account or login required to use AI features

✦ CUSTOMIZABLE AI
  - Bring your own GGUF model (Llama 3, Mistral, Phi-3, etc.)
  - Configurable GPU offload, context size, temperature, max tokens
  - Supports any instruction-tuned GGUF model
